# Python wrapper for Kiosker API
Python wrapper for Kiosker integration.

### Installation

```shell
pip3 install kiosker-python
```

### Setup

```python
from kiosker.api import KioskerAPI
api = KioskerAPI('10.0.1.100', 'ABC123')
```
